# BeanMap

A web app for discovering and organising cafés as points of interest (POI). Built for the WIT HDip in Computing Full Stack module (Assignment 1).

Locations can be categorised (e.g. best espresso, places to study) and managed with standard POI features: create, read, update, delete, and categories.

## Tech stack

- **Back end:** Node.js, Hapi, MongoDB (Mongoose)
- **Front end:** Handlebars
- **API:** REST, OpenAPI/Swagger, JWT auth
