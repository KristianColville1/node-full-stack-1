# BeanMap

A web app for discovering and organising cafés as points of interest (POI). Built for the WIT HDip in Computing Full Stack module (Assignment 1).

POIs have name and category; create, read, update, delete; sign up / log in.

## Tech stack

- **Back end:** Node.js, Hapi
- **Data:** In-memory store (facade for future DB levels)
- **Front end:** Handlebars, Bulma
- **API:** REST (basic)

## Usage

```bash
npm install
npm run dev
```

Open http://localhost:3000 (or the port shown). Sign up or log in, then use the dashboard to add and manage cafés.
